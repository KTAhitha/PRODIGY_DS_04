# PRODIGY_DS_04

## Data Science Internship - Task 04

This project is part of my Data Science Internship at Prodigy Infotech.

### Objective
Analyze and visualize sentiment patterns in social media data to understand public opinion and attitudes.

### Dataset
Twitter Training Dataset

### Tools & Technologies Used
- Python
- Pandas
- Matplotlib
- TextBlob

### Steps Performed
- Loaded the dataset
- Cleaned and processed text data
- Applied sentiment analysis using TextBlob
- Classified text into Positive, Negative, and Neutral sentiments
- Visualized sentiment distribution using a bar chart

### Output
- Sentiment classification of text data
- Bar chart showing distribution of sentiments

### Key Insights
- Majority of text data shows varying sentiments across categories
- Sentiment analysis helps understand public opinion effectively
- Positive, Negative, and Neutral sentiments can be clearly identified from text
